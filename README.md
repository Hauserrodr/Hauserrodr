### 🧠 What I do

My strength is the **ideas and the wiring**: connecting tools, automating pipelines, and getting
**bleeding-edge ML running reliably in production** — across vision, audio, 3D, and agents.

- 🔌 **Connecting tools & automating pipelines** — orchestrators, simulators, dataset builders, AutoML / semi-supervised loops, Airflow/Celery; I build the system that makes the *next* model easy.
- 🧪 **Bleeding-edge → production** — diffusion / flow-matching, self-supervised (BYOL/SimCLR), DINOv3 tracking, RLHF-inspired ranking, transformers, agents — shipped, not just read about.
- 🐷 **Published & validated** — cough-detection model **published in MDPI (2025)**, 99.6% field accuracy; commercial pilots with Cargill, BRF, Pamplona.
- 🛰️ **Edge & efficiency** — weight estimation on **300+ edge devices** (quantization, ONNX/TensorRT) — one proof that I make models cheap and reliable.
